Duplicate Image
Finder

Find and consolidate duplicate photos and videos on any platform. Local-only. Nothing leaves your machine.

git clone https://github.com/cozats/duplicate-finder.git && cd duplicate-finder

./start.sh

Features

Full-Resolution Preview

Click any thumbnail to view at full size. Use arrow keys to compare files side by side within a group.

Safe by Default

Files are moved to a trash folder, never deleted. Original timestamps are preserved. Nothing is permanent.

Memory Efficient

Files are hashed in 8MB chunks. Even multi-gigabyte video files never get fully loaded into memory.

Non-Blocking Scans

Scans run in a background thread with real-time progress. Cancel anytime without side effects.

Smart Defaults

The oldest file in each group is auto-selected as the keeper. One-click auto-resolve for all groups.

100% Local

No cloud. No database. No tracking. Everything runs on localhost. Your files never leave your machine.

How It Works

01

Enter one or more folder paths to scan

02

Every image and video is hashed with SHA-256

03

Files with identical hashes are grouped as duplicates

04

Pick a keeper for each group, or skip groups you want to leave alone

05

Consolidate moves duplicates to _duplicates_trash/ for review
